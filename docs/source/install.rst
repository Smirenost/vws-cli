Installation
------------

.. tabs::

   .. tab:: With ``pip``

      Requires Python 3.8+.

      .. code:: sh

         pip install VWS-CLI

   .. tab:: With Homebrew (macOS, Linux, WSL)

      Requires `Homebrew`_.

      .. code:: sh

         brew tap adamtheturtle/vws
         brew install vws-cli

   .. tab:: Linux binary

      Coming soon.
