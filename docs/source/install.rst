Installation
------------

With ``pip``
~~~~~~~~~~~~

Requires Python 3.8+.

.. code:: sh

   pip install VWS-CLI

With Homebrew (macOS, Linux, WSL)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Requires `Homebrew`_.

.. code:: sh

   brew tap adamtheturtle/vws
   brew install vws-cli

.. _Homebrew: https://docs.brew.sh/Installation
